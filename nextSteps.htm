Next Steps

There are many different components of a website that contribute to the quality and style of it. There are two other languages used in making websites which are Javascript and Cascade Style Sheets which can help to develop your webpage.

Javascript is used to add functionality to a website whereas Cascade Style Sheets is used to add style to it. Javascript is implimented in the <script> tag or as an attribute in a tag. Javascript can be used to change the content of your webpage based on your visitors interactions with certain elements. Javascript can even do math for you.

Cascade Style Sheets add style to your website which is very important to use when making professional webpages. It is implemented with the <style> tag or as an attribute in a tag. Cascade Style Sheets is abreviated as CSS and when used with HTML and Javascript you can make a professional looking and feeling webpage.
