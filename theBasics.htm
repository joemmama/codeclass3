The Basics

HTML stands for Hyper Text Markup Language and is the coding language used for making websites. It is made up of several different parts, such as tags, elements, and attributes.

Tags are the words inside angle brackets, such as <p> which is a paragraph tag used to write text.


<p>Hello World!</p>Would print Hello World! to your website!


Also when you open a tag, you must close it. There are two different ways to close a tag, which are different for certain tags. The most common way is to write the tag as a starting tag but with a slash after the first angled bracket. For example; <p>...</p>. The </p> is the closing tag. The other way to close a tag is to do it in one tag. For example; <img src="image.jpg">. The img tag is used for putting images on your website, but unlike the Paragraph tag there is only one tag.


An opening and closing tag indicate an element.

Elements are the building blocks of websites, and are the way you are reading this.

An element is a set of opening and closing tags. Inside of the tags is the element's content. The most common elements are the multiple sizes of headings, <h1> through <h6>, the <a> tag which is for an external link to either a website or file. Like This, as well as the <button> tag which creates a clickable button which can be used for many things, like a way to get to another website!

An atrribute is a property in a tag to provide context to the element about certain information of the element.

Some commonly used attribute is, the href attribute, which provides a hyperlink reference to an element, it is often used in the link tag. (<a>).

Another common attribute is src, the src attribute is used often in an image tag to point the computer to the source of the image you would like to see on your webpage.

Lastly, the id attribute is used to identify and modify certain groups of elements as long as their id is the same. It makes your coding process less repetitive, and inturn quicker.
